# File Structure

```   
    /project         
    |
    |
    |____/build             (the generated files/executables)
    |
    |
    |____/docs              (project documentation)
    |
    |
    |____/core              (core files / internal libraries)
    |    |
    |    |____/src          (all src files)
    |
    |
    |____/lib               (libraries for external exposure)
    |
    |
    |____/output            (output files from main app)
    |
    |
    |____/scripts           (scripts for running the application)
    |
    |
    |____/tests             (unit testing files)
    |
    |
    |____COPYING            (project license info coming soon)
    |
    |____INSTALL            (install instructions coming soon)
    |        
    |____README             (general readme of the project)
```
